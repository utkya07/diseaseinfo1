# CORONA_Info_page 🦠

A web-based informational portal that educates users about the COVID-19 (Coronavirus) pandemic. This project is created using **HTML**, **CSS**, and **JavaScript**, focusing on awareness, health safety, and preventive measures during the COVID-19 crisis.

---

## 📌 Features

- 🧾 Brief overview of COVID-19 (origin, timeline, transmission)
- 😷 List of common **symptoms**
- 🛡️ Preventive measures and **safety guidelines**
- 🧼 Hygiene practices (hand washing, mask usage, distancing)
- 📊 Optional: Real-time COVID-19 **statistics** using API
- 🎨 Responsive UI with clean and modern design
- ☑️ Interactive elements using JavaScript

---

## 🔧 Tech Stack

- **HTML5** – Structure of the page
- **CSS3** – Styling and layout
- **JavaScript** – Interactivity (e.g., dynamic content, collapsible sections, optional live data fetch)
